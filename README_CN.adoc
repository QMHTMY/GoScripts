== 描述
[link:README.adoc[English]] 几个go命令行脚本。

== 脚本
. QRcode - 以url生成二维码
. newCurl - 打印出给定网址的html源码
. ASCIIcode - 打印出给定字符的ascii值

== 使用
. 第一个 `./QRcode http://xxxx.xxx.xxx`
. 第二个 `./newCurl http://xxxx.xxx.xxx`
. 第三个 `./ASCIIcode xxx`
